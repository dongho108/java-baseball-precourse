# 미션 - 숫자 야구 게임

## 🚀 기능 목록
- 랜덤으로 1~9짜리 3개의 수를 생성한다.
- 사용자에게 수를 입력받는다.
- 입력받은 3자리 수에서 볼, 스트라이크 개수를 구해서 반환한다.
- 구해진 볼, 스트라이크를 통해 출력값을 결정한다.
  - 스트라이크 0개 : "낫싱"
  - 스트라이크 1개 ~ 2개 : "n볼 n스트라이크"
  - 스트라이크 3개 : "3스트라이크"
    - 정답문구 출력 : "3개의 숫자를 모두 맞히셨습니다! 게임 종료"
    - 계속 진행할 것인지 묻기 : "게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요."
      - 1 입력 : 처음으로 돌아간다.
      - 2 입력 : 프로그램을 종료한다.
